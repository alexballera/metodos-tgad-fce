Muestreo Sistemático¶

Si quisieramos hacer un muestreo rápido, puede incurrirse en un muestreo sistemático que toma una instancia cada $k$ observación.

n = 100  # cantidad de observaciones que queremos en la muestra
N = len(df) # tamaño de la población

k = N // n  # división entera

inicio = np.random.randint(0, k)
indices = np.arange(inicio, N, k) # Tomamos cada k-ésima observación
# indices = indices[:n]   # Ajuste por si nos pasamos de n para que solo tome esa cantidad de n, si querémos limitarlo.

muestra_sistem = df.iloc[indices]

El uso de conglomerados acá es didáctico, no necesario. Elegimos muestreos por conglomerados cuando trabajar a nivel individual es costoso o difícil. En este caso, las carreras funcionan como unidades naturales de agrupación. Esto simplifica el diseño y reduce costos, pero introduce mayor variabilidad respecto a un muestreo aleatorio.

Muestreo Estratificado¶

Hagamos un estratificado por género

In [90]:
n = 100  # tamaño deseado

# Calculamos proporción de cada género
proporciones = df['Gender'].value_counts(normalize=True) ## Normalize = True: nos ahorra un paso, devuelve proporciones. Sino devolvería frecuencias absolutas y deberíamos dividirla por el total.
n_por_estrato = (proporciones * n).round().astype(int)

diferencia = n - n_por_estrato.sum() # Ajuste por si hay diferencias por redondeo
if diferencia != 0:
    # Ajustamos al estrato más grande
    estrato_mayor = n_por_estrato.idxmax()
    n_por_estrato[estrato_mayor] += diferencia
